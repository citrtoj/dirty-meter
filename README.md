# dirty-meter
First Vue project. Uses Netlify Functions, the Spotify API and the lyrics.ovh API. Available [here](dirty-meter.netlify.app)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
